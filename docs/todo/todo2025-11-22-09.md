  - pnpm 另外提示：`Local package.json exists, but node_modules missing, did you mean to install?`，表示目前尚未安裝 node_modules。
- 初步結論：
  - 目前無法直接 `pnpm build`，主要原因是：
    - 依賴尚未安裝（node_modules 缺失）。
    - `cross-env` 尚未在本機 node_modules/.bin 中可用（應會隨依賴安裝一併解決）。
  - 下一步計畫是先執行 `pnpm install` 安裝所有 workspace 依賴，再重試 `pnpm build`。

  - Docker 啟動與服務驗證：
